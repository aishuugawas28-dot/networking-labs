# Router Network Lab

## Objective
To establish communication between two different networks using a router.

## Devices Used
- 1 Router (2911)
- 2 Switches (2960)
- 2 PCs
- Copper Straight-through Cables

## Network Configuration

| Device | IP Address |
|---|---|
| PC0 | 192.168.1.10 |
| Router G0/0 | 192.168.1.1 |
| Router G0/1 | 192.168.2.1 |
| PC1 | 192.168.2.10 |

## Concepts Learned
- Router functionality
- Routing basics
- Default gateway
- Inter-network communication
- Packet flow
