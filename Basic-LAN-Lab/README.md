# Basic LAN Lab

## Objective
To establish communication between two PCs using a switch in a LAN environment.

## Devices Used
- 1 Switch (2960)
- 2 PCs
- Copper Straight-through Cables

## IP Addressing

| Device | IP Address |
|---|---|
| PC0 | 192.168.1.1 |
| PC1 | 192.168.1.2 |

## Concepts Learned
- LAN communication
- Switch functionality
- IP addressing
- Ping testing
