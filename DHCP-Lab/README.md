# DHCP Server Configuration using Router

## Objective
To configure DHCP on a router and automatically assign IP addresses to client devices.

## Devices Used
- 1 Router (2911)
- 1 Switch (2960)
- 3 PCs

## Network Configuration

| Device | IP Address |
|----------|----------|
| Router G0/0 | 192.168.1.1 |
| PC0 | DHCP Assigned |
| PC1 | DHCP Assigned |
| PC2 | DHCP Assigned |

## Concepts Learned
- DHCP Configuration
- Dynamic IP Address Allocation
- DHCP Pool Creation
- Default Gateway Configuration
- Automatic Network Configuration
