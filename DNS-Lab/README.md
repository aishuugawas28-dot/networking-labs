# DNS Lab

## Objective
To configure and test DNS name resolution using a DNS server.

## Devices Used
- 1 Switch (2960)
- 1 PC
- 1 Server
- Copper Straight-through Cables

## Network Configuration

| Device | IP Address |
|---|---|
| PC0 | 192.168.1.2 |
| Server | 192.168.1.10 |

## DNS Configuration

| Domain Name | IP Address |
|---|---|
| www.myserver.com | 192.168.1.10 |

## Concepts Learned
- DNS basics
- Domain name resolution
- DNS server configuration
- Client-server communication
- Web server basics
